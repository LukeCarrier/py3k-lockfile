This is a port of lockfile 0.9.1 to Python 3.

It has been tested on Python 3.2, and may work on earlier 3.X as well.

All modifications are in the public domain, to the maximum extent possible
given the upstream code's license.  

Aaron Maxwell
Founder, Mobile Web Up

DEPENDENCIES

lockfile relies on minimock for tests.  As of this writing (March 2012),
minimock has a pending pull request on bitbucket for Python 3 support [0]
Until an official release happens, you can run Python 3 tests with the minimock
cloned from [0].

[0] https://bitbucket.org/t2y/minimock
